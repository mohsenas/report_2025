AI & Intelligent Automation

Nawras has evolved beyond traditional ERP by integrating artificial intelligence and advanced document processing capabilities.

AppAIAgent Framework

Established a comprehensive AI agent framework that evolved throughout 2025:

  • Built core infrastructure with gRPC services and agent configuration systems
  • Implemented specialized agents for Sales, Purchase, Inventory, and Risk Assessment
  • Developed lead time prediction using machine learning models
  • Implemented multi-tenant model training and execution
  • Created prescriptive recommendation systems

Document Processing

Advanced OCR capabilities supporting Arabic and English:

  • Integrated Tesseract OCR engine and ImageMagick preprocessing
  • Built document text extraction service with PDF support
  • Developed invoice parsing and supplier invoice matching
  • Created Docker support for all dependencies

Audit Service

  • Designed and implemented comprehensive audit logging system
  • Integrated across multiple modules with customizable auditing behavior
  • Added multi-tenancy support for audit records
  • Created REST and gRPC interfaces for audit log retrieval
