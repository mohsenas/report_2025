5. Platform & Infrastructure Modernization

The technological foundation of the Nawras ecosystem has been radically upgraded to ensure future-proof stability and developer efficiency.

UI/UX & Framework Upgrades

  • Angular 20: Migrated the entire platform to Angular 20, adopting modern control flows (@if, @for) and standalone component patterns.
  • Module Federation: Leveraged Micro-frontend architecture to scale the development of decoupled modules like Finance, Sales, and Warehouse.
  • Shared Component Library: Developed a suite of reusable UI components including Page Table Headers, Audit Log views, and dynamic Form Factories.

System Security & Quality

  • Global Audit Service: Designed a centralized audit logging framework with multi-tenancy support and gRPC/REST interfaces.
  • Testing Infrastructure: Established CI/CD pipelines with comprehensive unit and integration testing frameworks.
  • Performance: Optimized data retrieval using CommonDQueryDataset for dynamic transformation and aggregation of large datasets.
